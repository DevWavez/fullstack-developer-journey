# Phase 1 – Final Mini Project

## Interactive Task Tracker

### Project Goal

Build a simple interactive task tracker using HTML, CSS and JavaScript.

### Core Features (planned)

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persist tasks using localStorage

### Concepts used

- DOM manipulation
- Event handling
- State management
- localStorage
- Conditional rendering

### Project Structure

- index.html – structure
- style.css – layout & styling
- script.js – logic
