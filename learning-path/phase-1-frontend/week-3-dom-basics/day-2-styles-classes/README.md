## Goal

Learn how to interact with HTML elements dynamically by changing styles and toggling classes using JavaScript.

## Topics

- Accessing elements via `getElementById`
- Changing styles directly using `element.style`
- Adding, removing, and toggling CSS classes with `classList`
- Event listeners for buttons

## Exercises

1. Change text color by clicking buttons
2. Toggle a CSS class on an element
3. Combine inline styles and CSS classes for interactivity

## How it works

- **Red/Green buttons** change the text color dynamically
- **Toggle Highlight button** adds or removes the `highlight` class, which changes background and font weight
- Event listeners detect user clicks and update the DOM in real-time

## Notes

- Using `classList` is cleaner than modifying multiple inline styles
- `toggle` is ideal for effects like highlighting, dark mode switches, or showing/hiding elements
- Experiment by adding more buttons or classes to see live DOM updates
