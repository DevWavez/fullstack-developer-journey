## âœ… Key Takeaways

### HTML & CSS

- Semantic HTML improves accessibility and SEO
- External CSS separates style from structure
- Flexbox enables flexible and responsive layouts
- Media queries support mobile-first design

### Git

- Small, descriptive commits are essential
- Feature branches isolate work from main
- Merge conflicts are normal and must be resolved carefully
- Maintaining a clean `main` branch ensures production stability
